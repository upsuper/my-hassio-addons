## Add-ons

### [Navidrome](./navidrome)

Modern Music Server and Streamer compatible with Subsonic/Airsonic.

### [My NGINX](./my_nginx)

Fully custom NGINX instance.

## Installation

1. Navigate to **Settings** → **Add-ons** → **Add-on Store** in Home Assistant.
2. Click the three-dot menu in the top right and select **Repositories**.
3. Add this repository URL: `https://github.com/upsuper/my-hassio-addons`
4. The add-ons from this repository will now appear in the store.
